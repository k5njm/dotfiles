# dotfiles
> Using dotbot: https://github.com/anishathalye/dotbot

## Requirements:
- Linux / MacOS
- Git
- Python 3

## Usage:

### Setup a new machine:
```sh
# Clone into ~/.dotfiles
mkdir -p ~/repos &&  cd ~/repos && git clone https://github.com/k5njm/dotfiles dotfiles && cd dotfiles && ./install
```
